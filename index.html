Truck IDS

Game Simulasi Supir Truk Indonesia dengan Tantangan Realistis

Deskripsi Proyek

Truck IDS adalah game simulasi mengemudi truk dengan setting khas Indonesia. Pemain menjalankan misi pengantaran barang, menghadapi lalu lintas AI, mengelola bahan bakar, dan melakukan upgrade kendaraan demi efisiensi.

Fitur Utama

Engine On/Off System
Kontrol Truk: Maju, Mundur, Belok
AI Traffic Realistis
Pengiriman Barang antar Kota
Simulasi Bahan Bakar
Sistem Upgrade Truk
Lingkungan Dinamis (Siang/Malam)
UI Interaktif dan Navigasi GPS

Spesifikasi Teknis

  • Platform: Android
  • Engine: Unity (URP)
  • Bahasa Pemrograman: C#
  • Modeling: Blender, Mixamo
  • UI: Unity UI Toolkit / Canvas System

Monetisasi

  • Iklan Rewarded untuk bahan bakar atau upgrade
  • In-App Purchase untuk fitur premium
  • Kostumisasi visual truk (livery, aksesoris, lampu)

Roadmap Perkembangan

Tahap Fitur
Alpha Kontrol Dasar, Engine, Pengiriman Barang
Beta AI Traffic, Sistem Bahan Bakar, UI
Release Sistem Upgrade, Monetisasi, Polish Final
Fitur dan konten baru akan segera hadir. Stay tuned!
